קטלוג מוצרי תלת מימד – ריפוי בעיסוק

📋 כיצד מקבלים טוקן גישה? (פעם ראשונה בלבד)

  1. היכנס לאתר github.com עם חשבון המנהל
  2. לחץ על תמונת הפרופיל (פינה ימנית עליונה) ← Settings
  3. גלול למטה ← Developer settings ← Personal access tokens ← Tokens (classic)
  4. לחץ Generate new token (classic)
  5. תן שם (לדוג' "קטלוג"), סמן את הרשאה repo, ולחץ Generate token
  6. העתק את הטוקן שהופיע (הוא מתחיל ב־ghp_) ושמור אותו במקום בטוח

⚠️ משתמשים ב-Fine-grained token (מתחיל ב־github_pat_)?

  1. בRepository access — בחר את המאגר 3DPrinterCatalog (או "All repositories").
  2. בRepository permissions → Contents — חובה: Read and write (לא רק Read).
  3. בלי הרשאת כתיבה ל-Contents תופיע השגיאה: "Resource not accessible by personal access token".
🔒 הטוקן נשמר בדפדפן שלך בלבד ולא נשלח לאף שרת חיצוני. כל שינוי בקטלוג מתבצע ישירות מול GitHub.
